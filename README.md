<p>

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/aramelheni/easy-legal.git
   ```

2. Install dependencies:

   ```bash
   cd easy-legal
   npm install
   ```

3. Start the server:

   ```bash
   npm start
   ```
